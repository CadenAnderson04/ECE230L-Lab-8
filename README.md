## Lab Questions

### In plain English describe the function and use of a multiplexer.
  - A MUX is known as a 'Data Selector' & takes in multiple data inputs & produces a single output. The output is entirely dependant on the enable switch, if enable is 0/off the output is forced into an either all HIGH or LOW state, depending on the circuit. Another aspect to consider is the selector switch which selects which input is chosen depending on its value, with a 2-1 MUX, a selector value of 0/LOW selects the least significant input, while a 1/HIGH selector input selects the most significant input.
    
### In plain English describe the function and use of a demultiplexer.
  - A DeMUX is known as a 'Data Distributor' & takes in a single data input & outputs 2^n data lines. Also can utilize enable switches which allow outputs to represent themselves or forcing a neutral state for all.
